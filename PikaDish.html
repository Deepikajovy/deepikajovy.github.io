PikaDish (www.pikadish.com)

A Weekly Meal Planner Application

PikaDish is a personal project that I undertook with 3 other collegues. It is a mobile and web application that allows you to plan every meal you wish to cook in the given week, and then tally's a list of the required ingredients to cook these meals. As you can imagine, this is very helpful when planning your weekly shop.

To create this application, we used Asp.Net Web API with a SQL database in the backend, and Angular JS, JQuery and Bootstrap in the frontend.

To fully utilize the potential of this application, we set about converting it to a mobile application as well. My responsibilities included the front end design, Login module and the backend database design.
PhoneGap was used to create the mobile version of the application.
